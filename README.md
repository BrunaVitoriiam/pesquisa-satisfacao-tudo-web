# Pesquisa de Satisfação - Tudo Web

Projeto em Python que coleta a opinião de clientes e gera um relatório automático.

 🎯 Objetivo
Avaliar o atendimento com base nas avaliações de 5o clientes em:
- 1 = Excelente  
- 2 = Bom  
- 3 = Ruim  

 💻 Tecnologias
- Python

 📈 Resultado
O programa conta automaticamente as respostas usando estrutura de repetição FOR.
