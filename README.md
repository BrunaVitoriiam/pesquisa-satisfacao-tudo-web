## Pesquisa de Satisfação - Tudo Web

Projeto em Python que coleta a opinião de clientes e gera um relatório automático de satisfação com as porcentagem de cada avaliação.



## Objetivo
Avaliar o atendimento com base nas 50 avaliações realizada por cada pessoa, sendo as os graus de satisfação:

- 1 = Excelente  
- 2 = Bom  
- 3 = Ruim 


##💻 Tecnologias usadas
- Python online  
- Estrutura de repetição (for)  
- Estrutura condicional (if / elif)  
- Listas e dicionário
- IA


## Como funciona o programa

O sistema percorre a lista com os 50 entrevistados utilizando um laço de repetição (for).

Para cada cliente existe uma avaliação,  o programa verifica a opinião de cada um:

- Se for 1 → contabiliza como "Excelente"
- Se for 2 → contabiliza como "Bom"
- Se for 3 → contabiliza como "Ruim"

depois dessa verificação o programa:

- Soma todas as respostas
- Calcula o total de entrevistados automaticamente, que nesse caso foi 50 pessoas
- Exibe os resultados finais
- Calcula as porcentagens de cada categoria ( Excelente, bom, ruim)



## Resultado 
O programa exibe:

- Quantidade de respostas "Excelente"
- Quantidade de respostas "Bom"
- Quantidade de respostas "Ruim"
- Percentual de cada categoria
